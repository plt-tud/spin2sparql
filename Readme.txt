 Spin2Sparql
 ===========
 
 This project works with Eclipse and Maven. It converts SPARQL queries into SPIN models.
 
 Build JAR with ''mvn assembly:assembly''
 
 Run with ''java -jar spin2sparql-0.0.1-SNAPSHOT-jar-with-dependencies.jar''
